## To You Design の意味

**To You Design** は、「あなたへ届けるデザイン」という想いを込めたブランド名です。

| 単語 | 意味 |
|------|------|
| **To You** | あなたへ。見てくれる採用担当者・エンジニア・協業パートナーへのメッセージ |
| **Design** | 見た目のデザインだけでなく、UI/UX・情報設計・コード設計まで含む「ものづくり」全体 |

技術を積み上げるだけでなく、**誰かのために何を届けるか**を常に意識して開発する——それがこの名前の根底にある考え方です。

- **ユーザー視点**: 使いやすさ・読みやすさを最優先に設計する
- **相手への配慮**: 採用担当者が短時間で理解できる情報設計、エンジニアが技術力を把握できる構成
- **作り手の温かみ**: ロゴの鉛筆アイコンが象徴するように、手を動かして丁寧に仕上げる姿勢

「自分のスキルを見せる」だけでなく、「あなたにとって価値のあるものを届ける」——そのスタンスをサイト名に込めています。

## コンセプト

「作り手の温かみと技術力の融合」

エンジニアとしての技術力（Next.js, Three.js など）と、デザインへのこだわりを伝えるポートフォリオ。
清潔感のあるブルーを基調とし、モダンなタイポグラフィとインタラクティブな要素（3D・アニメーション）を組み合わせています。

## 色について

青色を基調とし、モダンで清潔感のあるデザインを目指しています。

| 用途 | 変数名 | 値 | 説明 |
|------|--------|-----|------|
| メインカラー | `$main-color` | `oklch(65% 50% 261.57)` / `#4485ff` | 見出し・ボタン・アクセント全般 |
| 下層ヒーロー背景 | — | `#4284ff` | About / Portfolio / Blog / Contact の帯 |
| 背景（セクション） | `$back-color` | `#f7f7f7` | Portfolio・Skill セクション |
| テキスト | `$black` | `oklch(16.01% 0 0)` | 本文・ナビゲーション |
| 白 | `$white` | `oklch(100% 0 0)` | カード背景・ヒーロー文字 |
| タグ（技術スタック） | `$tag-color-4` など | 5色パレット | 制作物の技術タグに使用 |

3Dモデル（Three.js）にもメインカラー `#4485ff` を適用し、画面全体のトーンを統一しています。

## タイポグラフィ

| 要素 | フォント | 用途 |
|------|----------|------|
| 本文 | Noto Sans CJK JP | 全ページ共通の読みやすい日本語 |
| 英字見出し（FV・セクション） | Futura PT Bold | 「To You Design」「Portfolio」などのインパクト |
| セクション見出し | `.main__title` | 英字大見出し + `data-ja` 属性で日本語サブタイトル |
| 下層ページ見出し | `.lower__title` | 白文字・青背景帯内の日英併記タイトル |

見出しは英字を大きく、日本語を小さく添える二層構造で、モダンさと可読性を両立しています。

## 共通UIパターン

- **ヘッダー**: ロゴ（鉛筆アイコン + To You Design）+ ナビ（About / ポートフォリオ / ブログ）+ お問い合わせボタン（青枠・矢印付き）
- **下層ヒーロー**: 高さ 250px の青帯（`#4284ff`）に白文字の日英タイトル
- **パンくず**: トップ > 現在ページ
- **共通ボタン**: 青枠 → ホバーで青塗り + 白文字（`.common__btn`）
- **スクロールアニメーション**: `ScrollComponent`（フェードイン）で各セクションを段階的に表示
- **フッター**: ロゴ + サイトマップリンク + © 年表記

## 各ページ

### トップページ

モダンなタイポグラフィと 3D を組み合わせたランディング構成。

| セクション | 内容 | デザインの特徴 |
|-----------|------|----------------|
| **ファーストビュー** | Three.js のジオデシック球体 + 「To You Design / Portfolio」 | 100vh 全画面。3Dモデルがゆっくり回転・上下に浮遊。Futura PT Bold の大見出しがモデル上に重なる |
| **About** | プロフィール写真・自己紹介・趣味・資格・SNSリンク | 2カラム（写真 / テキスト）。左寄せレイアウトで読みやすさ重視 |
| **Portfolio** | Swiper スライドで制作物プレビュー | グレー背景 + 波型デコレーション（`HomeWaveBg`）。カードにホバーズーム |
| **History** | 経歴タイムライン | 青い丸マーカーの縦タイムライン。About ページへの導線 |
| **Skill** | フロント / バック / インフラ / その他 | グレー背景。アコーディオン式カード（アイコン + スキル名 + 詳細） |
| **Contact** | お問い合わせへの導線 | 背景画像（`top_contact_bg.webp`）+ 白いカードボックス |

### Aboutページ

経歴・職歴・資格を体系的に伝えるプロフィール詳細ページ。

- **ヒーロー**: 青帯に「プロフィール / About」
- **タブ切り替え**: 「経歴」「職歴」をラジオボタン式タブで切替（アクティブタブは青上線）
- **タイムライン**: 日付 + タイトル + 説明文の縦並び（青丸マーカー）
- **資格セクション**: 日付・資格名のテーブル形式リスト

採用担当者がスキルとキャリアの流れを素早く把握できる情報設計。

### Portfolioページ

制作実績を一覧で見せるページ。

- **ヒーロー**: 青帯に「Portfolio / 制作物」
- **フィルター**: 右上に「並び替え」ドロップダウン（新着順 / 古い順）
- **カードグリッド**: 3カラム（SP は 1カラム）。サムネイル + 日付 + タイトル + 技術タグ（`#React` など）
- **カードホバー**: 画像が `scale(1.1)` でズーム。シャドウ付きカード

各制作物の詳細ページでは、概要・機能・こだわりポイントを画像付きで紹介。

### Blogページ

Zenn 連携の技術ブログ一覧ページ。

- **ヒーロー**: 青帯に「Blog / ブログ」
- **2カラムレイアウト**: 左に記事一覧（3カラムグリッド）、右に「おすすめ記事」サイドバー
- **記事カード**: 絵文字サムネイル + 日付 + タイトル + タグ（tech など）+ いいね数
- **ページネーション**: 青アクティブ状態の番号リンク（9件 / ページ）
- **SP**: サイドバー非表示、記事は 1カラム

エンジニア向けに技術記事の発信力をアピールする構成。

### Contactページ

お問い合わせフォームページ。

- **ヒーロー**: 青帯に「お問い合わせ / contact」
- **フォーム**: 名前・メールアドレス・メッセージ（2カラム: ラベル左 / 入力右）
- **バリデーション**: Zod + React Hook Form によるリアルタイムエラー表示
- **確認フロー**: 入力 → 確認画面 → EmailJS で送信
- **送信ボタン**: 青枠の「確認画面へ >」ボタン（共通ボタンスタイル）

採用・協業の問い合わせをスムーズに受け付けるシンプルな導線。

## インタラクション・技術要素

| 要素 | 技術 | 説明 |
|------|------|------|
| 3D ヒーロー | Three.js + React Three Fiber | GLTF モデルの回転・浮遊アニメーション。OrbitControls でマウス操作 |
| スライド | Swiper | トップのポートフォリオプレビュー |
| フェードイン | カスタム Hook (`use-fadeIn`) | スクロール連動の出現アニメーション |
| 波型背景 | CSS + WebP 画像 | セクション間の視覚的な区切り |
| ページ遷移 | `TransitionLink` | スムーズな画面切り替え |

## レスポンシブ対応

- **PC（1100px 以上）**: フルレイアウト。3カラムグリッド、2カラム About
- **タブレット（〜1024px）**: FV 高さ 70vh、スキル 2カラム、ポートフォリオ 2カラム
- **SP（〜580px）**: ハンバーガーメニュー（青ドロワー）、1カラム化、FV フォント縮小
