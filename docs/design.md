# Design System - To You Design Portfolio

## 1. コンセプト
「作り手の温かみと技術力の融合」
エンジニアとしての技術力（Next.js, Three.jsなど）と、デザインへのこだわりを伝えるポートフォリオ。
清潔感のあるブルーを基調とし、モダンなタイポグラフィとインタラクティブな要素（3D・アニメーション）を組み合わせています。

## 2. カラーパレット (Color Palette)
明度・彩度・色相を明示的に指定できる **OKLCH** を採用し、視覚的に統一された鮮やかな発色を実現しています。

| 色名 | 用途 | 値 (OKLCH / Hex相当) |
| :--- | :--- | :--- |
| **Main Color** | プライマリカラー、ヒーローテキスト、ボタン | `oklch(63.8% 48.5% 261.57)` (#4485ff) |
| **Main Disabled** | 非活性、サブ的な要素 | `oklch(55% 48.5% 261.57)` |
| **Black** | 本文テキスト、アイコン | `oklch(16.01% 0 0)` (#272727相当) |
| **White** | 背景、カード、テキスト（反転） | `oklch(100% 0 0)` (#ffffff) |
| **Grey** | ボーダー、サブテキスト | `oklch(54% 0 0)` |
| **Back Color** | ページ背景 | `#f7f7f7` |

### タグカラー (Tag Colors)
スキルのタグやカテゴリー分けに使用されます。
- **Tag 1 (Red)**: `oklch(60% 0.18 29.27)` (#d50303)
- **Tag 2 (Green)**: `oklch(60% 0.18 145.56)` (#00a339)
- **Tag 3 (Pink)**: `oklch(60% 0.18 327.74)` (#ce61c8)
- **Tag 4 (Dark Blue)**: `oklch(60% 0.18 251.34)` (#063997)
- **Tag 5 (Orange)**: `oklch(60% 0.18 45.73)` (#fd6d39)

## 3. タイポグラフィ (Typography)

### フォントファミリー
- **欧文/数字 (Heading)**: `futura-pt-bold, sans-serif`
  - 幾何学的でモダン、信頼感を与えるデザイン。
- **本文 (Body)**: `noto-sans-cjk-jp, sans-serif`
  - 日本語の可読性を重視したスタンダードなゴシック体。

### タイポグラフィスケール
- **Hero Title**: `120px` (SP: `34px`)
- **Section Title**: `60px` (SP: `34px`)
- **Sub Title**: `20px`
- **Body**: `16px` (Base)
- **Small**: `14px` (Button, Tags)

---

## 4. スペーシング & グリッド (Layout)

### コンテナ
- **Max Width**: `1100px`
  - 左右のマージンは画面サイズに応じて調整。
- **Responsive**:
  - `PC`: 固定幅コンテナ
  - `Lower than 1024px`: `90%` 幅
  - `Lower than 580px (SP)`: `90%` 幅、要素を縦にスタック

### 余白ルール
- セクション間の余白: `60px` 〜 `80px`
- 要素内のパディング: `rem()` 関数を使用してベースフォント（16px）基準で算出。

---

## 5. UIコンポーネント (Components)

### ボタン (Buttons)
- **スタイル**: 角丸（8px）、ボーダー（1.5px）。
- **ホバー**: 
  - 背景色がプライマリカラーに変化し、テキストが白反転。
  - スライドエフェクト（擬似要素 `::before` を使用した左からの塗りつぶし）を採用。

### カード (Cards)
- ポートフォリオ一覧、ブログ一覧に使用。
- 画像を上部に配置し、下部にテキストとタグを配置。
- 控えめな `box-shadow` とホバー時の `opacity` 変化。

### ナビゲーション
- 固定ヘッダー。
- SP版ではフルスクリーンオーバーレイのハンバーガーメニュー。

---

## 6. インタラクション & 表現 (Visual Effects)

### 3D 表現 (Three.js)
- ヒーローセクションに配置された **Red Triangular Cage Sphere**。
- `React Three Fiber` を使用し、マウス追従や回転を行うことで「モダン・先進性」を表現。

### アニメーション
- **Page Transitions**: `framer-motion` の代わりにカスタムCSSアニメーションを使用。
- **Scroll Behavior**: `smooth` 指定。
- **Micro Interactions**: リンクやボタンへのホバー時に `0.7` の不透明度変化。
